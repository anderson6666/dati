## 1. 产品概述

全网智能题库总结大师是一款部署在 GitHub Pages 的静态智能备考工具，面向备考学生与考证学员。前端调用知乎搜索 API 采集全网分散的真题回忆、网友题库与答题经验，再交由 Agnes 大模型完成去重、标准化与应试技巧提炼，生成全网覆盖最完整的结构化题库，并支持在线刷题与文件导出，实现一站式刷题备考。

* 解决痛点：网络资料分散杂乱、题库重复缺失、缺少针对性应试技巧

* 目标用户：刷题应试的人

* 价值意义：极大降低考生整理资料与总结考点的时间成本，依托 GitPages 免费部署，浏览器打开即可使用

### 2.2 功能模块

1. **首页**：Hero 搜索区、四大核心功能展示、热门考试快捷入口、数据统计概览
2. **采集中心**：考试关键词输入、知乎 API 与 Agnes API 配置、采集进度可视化、原始素材预览
3. **题库大厅**：题目卡片列表、知识点/题型/难度多维筛选、关键词搜索、题目详情展开
4. **刷题面板**：单题沉浸式答题、即时反馈、应试技巧绑定展示、答题进度与统计
5. **应试技巧**：按题型/知识点分类的秒杀口诀、避坑指南、记忆方法卡片
6. **错题本**：错题标记管理、错题重做模式、掌握度统计
7. **导出中心**：JSON 完整题库导出、PDF 资料导出、自定义导出范围

## 3. 核心流程

**主流程**：用户输入考试关键词 → 调用知乎搜索 API 抓取全网素材 → 调用 Agnes API 去重标准化 → Agnes 生成应试技巧 → 题库入库 → 用户在线刷题 → 标记错题 → 导出复习资料

**刷题流程**：选择考试/知识点 → 进入刷题面板 → 逐题作答 → 提交查看正误与解析 → 查看绑定技巧 → 错题自动入本 → 完成后查看统计

```mermaid
    "B --> C[抓取全网素材]"

    "D --> E[Agnes生成应试技巧]"
    "E --> F[题库入库展示]"
    "F --> G[用户在线刷题]"
    "G --> H[答题反馈与技巧展示]"
    "H --> I[错题自动入本]"
    "I --> J[导出JSON/PDF复习资料]"
    "F --> K[技巧中心浏览]"
```

## 4. 用户界面设计

### 4.1 设计风格

* **美学方向**：学术编辑风（Editorial Scholarly）——高端学术期刊与现代知识终端的融合，沉稳克制中带有知识殿堂的庄重感

* **主色调**：深墨黑 `#1a1612`（背景）、暖羊皮纸 `#f5f0e8`（卡片）、烧琥珀金 `#c8862c`（强调）、深酒红 `#8b2635`（次强调）

* **按钮风格**：扁平微圆角（4px），主按钮深墨底配琥珀金文字，次按钮描边款

* **字体**：标题用 Cormorant Garamond（衬线显示字体，学术感），正文用 Noto Serif SC（中文衬线），数据/代码用 JetBrains Mono

* **布局风格**：编辑杂志式排版，强字体层级，慷慨留白与受控密度并存，卡片式题目展示

* **图标风格**：线性极简图标（lucide-react），与衬线字体形成现代与古典的张力

* **纹理细节**：羊皮纸纹理叠加、细微颗粒噪点、金色细线分隔

### 4.3 响应式

* 桌面优先设计（1280px+ 完整体验）

* 平板自适应（768-1280px 侧栏折叠为抽屉）

* 移动端适配（<768px 单列布局，触摸优化，底部导航栏）

* 触摸优化：按钮最小 44px 触摸区，滑动手势支持

### 4.4 3D 场景指引

本项目不涉及 3D 场景，以 2D 编辑风排版与微动效为主。
